# PicoGo for kids
The goal is to build a toy which teaches programming concepts to kids for the first time. Built based on observing how our 3 year old interacts with it :)

## Features

```
2: go forward
4: go left
6: go right
8: go back
-: decrease speed
+: increase speed
1: play simple song
3: play happy song
7: change LED color to blue
9: change LED color to red
Play: repeat steps what the kid just "programmed"
CH+: clear recorded steps
```

## Setup

Download and install Thonny. Connect device and copy main.py and included files to the device.

Based on https://www.waveshare.com/wiki/PicoGo#Demos
